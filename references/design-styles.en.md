## 1. Information Architecture School (01-04)
> Philosophy: "Data is not decoration, it's building material"

### 01. Pentagram - Michael Bierut Style
**Philosophy**: Typography is language, grid is thought
**Core features**:
- Extreme color restraint (black/white + 1 brand color)
- Modern interpretation of Swiss grid system
- Typography as primary visual language
- Strategic use of negative space (60%+ whitespace)

**Prompt DNA**:
```
Pentagram/Michael Bierut style:
- Extreme typographic hierarchy, Helvetica/Univers family
- Swiss grid with precise mathematical spacing
- Black/white + one accent color (#HEX)
- Information architecture as visual structure
- 60%+ whitespace ratio
- Data visualization as primary decoration
```

**Representative work**: Hillary Clinton 2016 campaign identity
**Search keywords**: pentagram hillary logo system

## Prompt Usage Instructions

**Combination formula**: `[Style Prompt DNA] + [Scene Template (see scene-templates.md)] + [Specific Content]`

### Core Principle: Describe Mood, Not Layout

Key to AI image generation: short prompts > long prompts. Describing 3 sentences of mood and content works better than 30 lines of layout details.

| What kills diversity | What激发 creativity |
|----------------|----------------|
| Specifying color ratios (60%/25%/15%) | Describing mood ("warm like Sunday morning") |
| Dictating layout positions ("title centered, image right") | Referencing specific aesthetics ("Pentagram editorial feel") |
| Restricting character poses and expressions | Letting AI naturally interpret style |
| Listing all visual elements to include | Describing what audience should feel |

### Good / Bad Examples

**Bad — Over-constrained (AI generates flat, empty):**
```
Professional presentation slide. Dark background, light text.
Title centered at top. Two columns below. Left column: bullet points.
Right column: bar chart. Colors: navy 60%, white 30%, gold 10%.
Font size: title 36pt, body 18pt. Margins: 40px all sides.
```

**Good — Mood-driven (generates diverse, textured):**
```
A data visualization that feels like a Bloomberg Businessweek
editorial spread. The key number "28.5%" should dominate the
composition like a headline. Warm cream tones with sharp black
typography. The data tells a story of dramatic channel shift.
```

### Execution Path Selection

According to quick reference "Best Path" column:
- **AI Generation**: Styles with clear visual elements (06/07/12/13/14/15/16/20), use Gemini/Midjourney direct output
- **HTML Rendering**: Styles relying on precise typography (01/03/04/10/11/17/18), code controls data and layout
- **Hybrid**: HTML skeleton layout + AI generated illustrations/backgrounds (02/05/08/09/19)

### Quality Control

1. ❌ Don't write "in the style of Pentagram" directly → ✅ Describe with specific design features
2. Text often errors in AI generation → Replace text after generating
3. Ratios easily distort → Specify aspect ratio explicitly
4. Generate 3-5 variants first, select best, then refine

**Default aesthetic forbidden zones** (user can override with own brand):
- ❌ Cyber neon/dark blue background (#0D1117)
- ❌ Personal signature/watermark on cover images
