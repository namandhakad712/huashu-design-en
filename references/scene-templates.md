## 1. WeChat Public Account Cover / Article Header

**Specifications**:
- Cover image: 2.35:1 (900×383px or 1200×510px)
- Body illustrations: 16:9 (1200×675px) or 4:3 (1200×900px)

**Key design elements**:
- Visual impact priority (user scans quickly in feed)
- Minimal or no text (WeChat title overlays)
- Moderate color saturation (WeChat reading environment is white)
- Avoid over-detail (must remain recognizable as thumbnail)

**Recommended styles**: 01 Pentagram / 11 Build / 12 Sagmeister / 18 Kenya Hara / 07 Field.io

## 2. Body Illustrations / Concept Art

**Specifications**:
- 16:9 (1200×675px) most common
- 1:1 (800×800px) good for emphasis
- 4:3 (1200×900px) good for information-dense

**Key design elements**:
- Serves article argument, not decoration
- Creates visual rhythm with context
- Expresses one core concept simply
- AI generation preferred, HTML screenshots only for precise data tables

## 3. Infographics / Data Visualization

**Specifications**:
- Vertical long format: 1080×1920px (mobile reading)
- Horizontal: 1920×1080px (embedded in articles)
- Square: 1080×1080px (social media)

**Key design elements**:
- Clear information hierarchy (title → key data → details)
- Accurate data, don't fabricate
- Visual guide lines (user reading path)
- Appropriate use of icons/charts to aid comprehension

**Recommended styles**: 04 Fathom / 10 Müller-Brockmann / 02 Stamen / 17 Takram

**Scene prompt template**:
```
[Insert style DNA here]
- Infographic / data visualization
- [Vertical 1080x1920 / Horizontal 1920x1080 / Square 1080x1080]
- Clear information hierarchy: title → key data → details
- Visual flow guiding reader's eye path
- Icons and charts for comprehension
- Data-accurate, no decorative distortion
```

## 4. PPT / Keynote Presentation

**Specifications**:
- Standard: 16:9 (1920×1080px)
- Widescreen: 16:10 (1920×1200px)

**Key design elements**:
- One core message per slide (don't pile)
- Clear type hierarchy (title 40pt+ / body 24pt+ / notes 16pt+)
- Generous whitespace, clearer in projection
- Image-to-text ratio at least 60:40
- Consistent visual system (colors, fonts, spacing)

**Recommended styles**: 01 Pentagram / 10 Müller-Brockmann / 11 Build / 18 Kenya Hara / 04 Fathom

**Scene prompt template**:
```

- Presentation slide design, 16:9
- One core message per slide

```
