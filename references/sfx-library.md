## Usage Guidelines

### Volume recommendations (from apple-gallery-showcase.md audio dual-track)
- **SFX main track**: `1.0` (no attenuation)
- **BGM background track**: `0.4~0.5` (SFX clearly penetrates)
- **Multiple SFX overlay**: use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### ffmpeg stitching templates
```bash
# Single SFX aligned to timepoint:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# Multiple SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection decision tree
1. **Has tactile action** (typing/click/swipe) → `keyboard/` or `ui/`
2. **Element enter/exit** → `transition/`
3. **Container layer operation** (cards/modal) → `container/`
4. **State feedback** (success/failure/notification) → `feedback/`
5. **Progress/time passage** → `progress/`
6. **Brand impact/moment** → `impact/`
7. **AI magic/transformation** → `magic/`
8. **Command line/code demo** → `terminal/`

### Avoid sound stacking
- Max 2 SFX concurrent at same timestamp
- Can have 3 when BGM drops below 0.3
- For brand impact, clear other SFX (leave 0.2s gap before landing)

## Prompt Writing Principles (for reuse)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**Three elements of good prompts**:
1. **Sound physical description**: what object, what action ("mechanical keyboard single key press")
2. **Texture/style qualification**: apple-style / ios-style / cinematic / retro
3. **Negative examples**: no reverb / clean studio / minimal

❌ "click sound"
✅ "crisp ui button click, clean modern interface sound, apple-style, high pitched"

❌ "magic"
✅ "bright twinkling stars sound, high pitched glittery chime, fairy dust"
